2. It is highly recommended to run with multiple processors, as more processors will finish the running faster.

4. Usage: "mpiexec -n numprocs python -m mpi4py pyfile". Please refer to https://mpi4py.readthedocs.io/en/stable/mpi4py.run.html for more details.
